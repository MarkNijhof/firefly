Firefly by ariejan

FireFly is a simple URL shortner for personal use.

Firefly is Rack based application, meaning that you can easily deploy it to Heroku or a web server using Passenger. Some servers (including Passenger) require you have a public directory available. Simple create an empty public directory and you're good to go.

Install

gem install firefly
