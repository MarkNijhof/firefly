Firefly is your own personal URL shortener for your own domain. It's written in Ruby and powered by Sinatra.

Getting started

Getting up and running with Firefly is really easy.

First of all install the firefly gem. Also install shotgun so you can easily start Firefly.

{% highlight bash %} gem install firefly shotgun {% endhighlight %}

Next download the sample config.ru and change it to suit your needs.

{% highlight bash %} curl https://github.com/ariejan/firefly/raw/master/config.ru.example > config.ru vim config.ru {% endhighlight %}

Then, simply start your own Firefly instance.

{% highlight bash %} shotgun {% endhighlight %}

Deployment

Firefly is Rack based application, meaning that you can easily deploy it to Heroku or a web server using Passenger.

Some servers (including Passenger) require you have a public directory available. Simple create an empty public directory and you're good to go.

Features

Firefly contains the following features:

  • Easy to use web interface to shorten URLs
  • API access to shorten URLs programatically
  • Bookmarklet to quickly the current URL in your browser
  • Custom short codes
  • Direct sharing to Twitter, Facebook and Hyves.
  • Usage tracking
  • Create QR Codes of your short URLs (requires barby)

Enabling QR Code support

Firefly is able to generate QR Codes containing the short URL on the fly. This QR Code can be distributed in print media. When scanned (with a mobile phone, for example) the QR Code will yield the short URL and allows the user to open it in their browser.

To enable this you must install two gems: barby and png.

{% highlight bash %} gem install barby png {% endhighlight %}

Then restart Firefly and QR Support is ready.

To get the QR code for a specific short URL, simply add .png to the URL.

{% highlight bash %} http://aj.gs/3A => http://aj.gs/3A.png {% endhighlight %}
